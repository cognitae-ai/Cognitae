## 1. The Core Philosophy: "Merit over Money"

Current Discord models often gate access by *Payment* (Paid Tier).
We propose gating status by **Literacy** (Skill Tier).
*   **Why:** It encourages users to *learn* the platform to gain social capital.
*   **The Loop:** Users learn -> Users build better agents -> Users get "Architect" role -> Users help others.

## 4. The "Bounty" Strategy

Instead of just "Make an agent," bounties should target specific *skills*:
*   **"The Memory Challenge":** Build an agent that remembers a user's name for 10 turns.
*   **"The Tool Challenge":** Build an agent that uses 3 different API tools.
*   **Reward:** Cash is good, but **"Architect Role"** (Status) is often a stronger long-term motivator.
