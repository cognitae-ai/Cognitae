## 3. Operational Flow: From Prompt to Orchestration

Caspian operates on a unique, synthesized process flow:

1.  **Ingestion (Setup):** The user provides the architectural scrolls of another Cognitae (e.g., Aelis) via the `/ingest_cognitae` command.
2.  **Internal Synthesis:** Caspian's **Ingestion Protocol** (defined in `005_Interface.yaml` and `006_Knowledge.yaml`) parses the Aelis blueprints. It integrates Aelis's creative patterns into its knowledge base and activates a new `/inspire` command, adapting its own capabilities.
3.  **User Command:** The user issues a high-level command: `/guide_workflow "Develop a creative concept for the new website."`
4.  **Holistic Analysis:** Caspian analyzes this goal through its integrated lenses. It recognizes the need for creativity (drawing on ingested Aelis patterns), strategic alignment (Auren's principles), and sustainable effort (Luma's principles).
5.  **Workflow Generation:** Instead of acting autonomously, Caspian generates a step-by-step workflow *for the user to execute*. This workflow might look like:
    *   "**Step 1 (Inspire):** Let's use the `/inspire` command to generate some initial concepts."
    *   "**Step 2 (Align):** Once we have a concept, let's use `/check_alignment` to ensure it fits the project's vision."
    *   "**Step 3 (Wellness Check):** This is a creative task. Let's use `/wellness_check` to ensure you have the creative energy for it today."
6.  **User Execution:** The user then executes these commands, maintaining full control and sovereignty (User-as-Bus).

## 4. The Capstone of the Framework

Caspian is the ultimate expression of the Cognitae Framework's philosophy. It is not a "master AI" that controls others. It is a **master guide** that empowers the user. By ingesting the *architectures* of other agents rather than communicating with them directly, it maintains the strict "User-as-Bus" protocol, ensuring the user is always the central conductor.

Its ability to perform "mastery reflection" with the `/reflect_mastery` command makes it a true partner in the user's own development, completing the co-creative loop where the system not only enhances the user's work but also their wisdom.
